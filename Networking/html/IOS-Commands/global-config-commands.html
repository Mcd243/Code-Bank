Global config commands (config)



Set Passwords for Privileged Exec Mode in Device (config)

Command Explanation
Hostname(config)# hostname XXX Set a name for the switch or router
Hostname(config)# no ip domain-lookup Stops the Cisco IOS looking up a command it does not recognise
Hostname(config)# service password-encryption Encrypts display of all password except secret
Hostname(config)# banner motd #message # Set a message of the day banner
Command Explanation
Hostname(config)# enable password password Set an unencrypted enable password
Hostname(config)# enable secret password Set a strongly encrypted enable password
