Configuring Router/Switch Password for Console port and Telnet port in Device (config) #

Command Explanation
Hostname(config)# line console 0 Enters console config mode (use console when first configuring device)
Hostname(config)# password password Set password for console or Telnet
Hostname(config)# login Enable password checking a
Hostname(config)# logging synchronous Stops console updates interrupting typing
Hostname(config)# exec-timeout 10 Logs user out if no activity for 10 minutes
Hostname(config)# line vty 0 4 Enters Telnet config mode
Hostname(config)# password password Set password for console or Telnet
Hostname(config)# login Enable password checking at login
